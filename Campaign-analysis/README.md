# Campaign Performance Analysis

## Overview
This project focuses on analyzing marketing campaign performance across multiple platforms including Instagram, Facebook, and YouTube.

## Objective
To evaluate campaign effectiveness and identify opportunities for improving engagement and conversions.

## Tools Used
- Microsoft Excel
- Power BI

## Key Metrics
- Click Through Rate (CTR)
- Conversion Rate
- Return on Investment (ROI)
- Engagement Rate

## Approach
The dataset was cleaned and transformed using Excel and Power Query. Key performance metrics were calculated and visualized using Power BI dashboards to identify patterns and trends.

## Key Insights
- Instagram Reels showed higher engagement compared to other platforms
- Influencer campaigns resulted in better conversion rates
- Video content performed better than static posts

## Business Recommendations
- Focus more on short-form video content
- Increase influencer collaborations
- Optimize budget allocation towards high-performing channels
